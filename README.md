# Run example project

1. Install dependencies:
```bash
cd MobDev && yarn
```
2. Lint:
```bash
yarn lint
```

3. Start the development server:
```bash
yarn start
```

When using Expo Go, ensure that both the mobile device and the PC are connected to the same Wi-Fi network.
