# Libraries

- [yarn add styled-components](https://classic.yarnpkg.com/en/package/styled-components): CSS-in-JS styling with support for theming and reusable styled components.

- [yarn add react-native-paper](https://reactnativepaper.com/): UI component library (material design).

- [yarn add react-native-safe-area-context](https://docs.expo.dev/versions/latest/sdk/safe-area-context/): To handle safe area insets and display content correctly within the notch and corners.

- [yarn add react-native-vector-icons](https://classic.yarnpkg.com/en/package/react-native-vector-icons): Icon library.

- [yarn add react-native-svg](https://www.npmjs.com/package/react-native-svg): SVG rendering.

- [yarn add @react-navigation/native](https://reactnavigation.org/docs/getting-started/): System's navigation.

- [yarn add @react-navigation/bottom-tabs](https://reactnavigation.org/docs/bottom-tab-navigator/): System's bottom tabs.

- [yarn add @react-navigation/stack](https://reactnavigation.org/docs/stack-navigator/): Navigation stack.

- [yarn add react-native-swiper-flatlist](https://www.npmjs.com/package/react-native-swiper-flatlist): Carousel/swiper.

- [yarn add react-native-gifted-chat](https://www.npmjs.com/package/react-native-gifted-chat?activeTab=versions): Chat UI components.

## Type Definitions

- [yarn add @types/styled-components](https://www.npmjs.com/package/@types/styled-components): Type definitions for styled-components.

# Expo SDK Modules

- [npx expo install expo-av](https://docs.expo.dev/versions/latest/sdk/av/): Audio and video playback.

- [npx expo install expo-font](https://docs.expo.dev/versions/latest/sdk/font/): Text fonts.

- [npx expo install @expo-google-fonts/oswald](https://www.npmjs.com/package/@expo-google-fonts/oswald): Prepackaged Google Font (uses expo-font under the hood).

- [npx expo install expo-image-picker](https://docs.expo.dev/versions/latest/sdk/imagepicker/): Access camera & media library.

- [npx expo install react-native-gesture-handler](https://docs.expo.dev/versions/latest/sdk/gesture-handler/): Gesture system (tap, swipe, pan, etc.).

- [npx expo install react-native-reanimated](https://docs.expo.dev/versions/latest/sdk/reanimated/): Advanced animation engine.

- [npx expo install react-native-screens](https://docs.expo.dev/versions/latest/sdk/screens/): Native screen optimization for navigation.

- [npx expo install expo-location](https://docs.expo.dev/versions/latest/sdk/location/): Device's geolocation information.
